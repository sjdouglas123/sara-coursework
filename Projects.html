My Project's

Database Project

In Higher Computing for our assessed assignment I created a database using Microsoft Access and then read this database into a Visual Basic program, I then had to manipulate the data to get certain parts as output until it worked. I really enjoyed this assignment and I recieved a mark of 59/60 in it.

Car project - Arduino & Processing

I am currently involved in a project with 4 other people where we are using Arduino and Processing to control a toy car. We have got a camera on a phone to work with processing, this will be strapped to the car so that we can remotely drive it around the room. We also have programmed a buzzer that will act as the horn, once a key has been pressed processing will send signals to arduino to activate the buzzer horn. This is currently in process and is expected to be finished by the 10th of December 2017.
